# Kata: Discover Commands with the LazyVim Picker

## Task

Practice finding an Ex command without typing command-line completion syntax.

## Start

Start in Normal mode in LazyVim. The cursor position does not matter.

## End

The command picker should be closed after confirming that `colorscheme`
appears in it.

## Commands

Run these command steps:

```text
1. <leader>sC
2. colorscheme
3. <Esc>
```
