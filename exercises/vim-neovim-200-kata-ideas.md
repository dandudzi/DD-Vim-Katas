# 200 Vim and Neovim Kata Ideas

This catalog contains exactly 200 distinct motions, editing techniques, and workflows. Each entry includes the key sequence or command, its purpose, and a concrete kata prompt. The list was researched with Exa from Vim/Neovim help and upstream GitHub documentation, then normalized to remove duplicate techniques.

## Core motions, searches, jumps, and text objects

1. **Counted basic motion** - Keys: `3h`, `4j`, `2k`, `5l`. Move an exact character or line count. Kata: land on marked cells without arrow keys. [Source](https://vimhelp.org/motion.txt.html)
2. **Physical line start** - Key: `0`. Jump to column zero, including indentation. Kata: remove leading indentation from selected lines. [Source](https://vimhelp.org/motion.txt.html)
3. **First non-blank** - Key: `^`. Jump to the first non-whitespace character. Kata: edit the first token on differently indented lines. [Source](https://vimhelp.org/motion.txt.html)
4. **Line end** - Key: `$`. Jump to the last character; a count can move down lines. Kata: append punctuation to several target lines. [Source](https://vimhelp.org/motion.txt.html)
5. **Last non-blank** - Key: `g_`. Reach the last non-whitespace character. Kata: change trailing tokens while preserving spaces. [Source](https://vimhelp.org/motion.txt.html)
6. **Exact screen column** - Key: `{count}|`. Jump to screen column `{count}`. Kata: edit fixed-width records at columns 10, 20, and 30. [Source](https://vimhelp.org/motion.txt.html)
7. **Find forward on a line** - Key: `f{char}`. Land on the next or count-th character. Kata: delete through a chosen delimiter with `df,`. [Source](https://vimhelp.org/motion.txt.html)
8. **Till forward on a line** - Key: `t{char}`. Land immediately before the target. Kata: change text up to a colon with `ct:`. [Source](https://vimhelp.org/motion.txt.html)
9. **Backward character hunt** - Keys: `F{char}`, `T{char}`. Find backward on, or just after, a character. Kata: repair a prefix while navigating from line end. [Source](https://vimhelp.org/motion.txt.html)
10. **Repeat a character hunt** - Keys: `;`, `,`. Repeat the latest `f/F/t/T` in the same or opposite direction. Kata: visit every comma from one initial search. [Source](https://vimhelp.org/motion.txt.html)
11. **Next word start** - Key: `w`. Move by keyword-style words. Kata: reach a requested identifier with a count. [Source](https://vimhelp.org/motion.txt.html)
12. **Next WORD start** - Key: `W`. Move by whitespace-delimited WORDs. Kata: navigate punctuation-heavy paths. [Source](https://vimhelp.org/motion.txt.html)
13. **Word end** - Key: `e`. Move to the end of a word. Kata: uppercase through the third word end. [Source](https://vimhelp.org/motion.txt.html)
14. **WORD end** - Key: `E`. Move to the end of a whitespace-delimited WORD. Kata: operate through an entire URL or dotted value. [Source](https://vimhelp.org/motion.txt.html)
15. **Previous word start** - Key: `b`. Move backward by words. Kata: return from a suffix to an identifier start. [Source](https://vimhelp.org/motion.txt.html)
16. **Previous WORD start** - Key: `B`. Move backward by WORDs. Kata: traverse a punctuation-rich command backward. [Source](https://vimhelp.org/motion.txt.html)
17. **Previous word end** - Key: `ge`. Move backward to the preceding word end. Kata: land on the prior token's final character. [Source](https://vimhelp.org/motion.txt.html)
18. **Previous WORD end** - Key: `gE`. Move backward to the preceding WORD end. Kata: navigate backward among whitespace-separated arguments. [Source](https://vimhelp.org/motion.txt.html)
19. **Display-line movement** - Keys: `gj`, `gk`. Move through wrapped screen lines rather than logical lines. Kata: navigate a wrapped paragraph without changing settings. [Source](https://vimhelp.org/motion.txt.html)
20. **Adjacent line first non-blank** - Keys: `+`, `-`. Move down or up and land on the first non-blank. Kata: visit code statements while ignoring indentation. [Source](https://vimhelp.org/motion.txt.html)
21. **Count-relative first non-blank** - Key: `{count}_`. Move count minus one lines down to first non-blank. Kata: reach the fourth following statement with one motion. [Source](https://vimhelp.org/motion.txt.html)
22. **Start or numbered line** - Keys: `gg`, `{count}gg`. Go to line one or an exact line. Kata: navigate directly to supplied line numbers. [Source](https://vimhelp.org/motion.txt.html)
23. **End or numbered line** - Keys: `G`, `{count}G`. Go to the final line or an exact line. Kata: jump to file end, then line 17. [Source](https://vimhelp.org/motion.txt.html)
24. **Percentage jump** - Key: `{count}%`. Jump to a percentage through the buffer. Kata: visit the 25%, 50%, and 75% positions. [Source](https://vimhelp.org/motion.txt.html)
25. **Byte-offset jump** - Key: `{count}go`. Go to an exact byte offset. Kata: locate markers using provided byte positions. [Source](https://vimhelp.org/motion.txt.html)
26. **Viewport-relative landing** - Keys: `H`, `M`, `L`. Jump to the top, middle, or bottom visible line. Kata: edit three visible targets without searching. [Source](https://vimhelp.org/motion.txt.html)
27. **Reposition the cursor line** - Keys: `zt`, `zz`, `zb`. Put the current line at viewport top, center, or bottom. Kata: inspect context above and below one line. [Source](https://vimhelp.org/scroll.txt.html)
28. **Fine viewport scroll** - Keys: `<C-e>`, `<C-y>`. Scroll one line without initially moving the cursor. Kata: reveal hidden context while preserving cursor location. [Source](https://vimhelp.org/scroll.txt.html)
29. **Half-page scroll** - Keys: `<C-d>`, `<C-u>`. Scroll down or up roughly half a window. Kata: scan a long file while maintaining context. [Source](https://vimhelp.org/scroll.txt.html)
30. **Full-page scroll** - Keys: `<C-f>`, `<C-b>`. Move forward or backward one screen. Kata: locate a heading using page-sized movement only. [Source](https://vimhelp.org/scroll.txt.html)
31. **Sentence motion** - Keys: `(`, `)`. Move to previous or next sentence boundary. Kata: delete or reformat one prose sentence. [Source](https://vimhelp.org/motion.txt.html)
32. **Paragraph motion** - Keys: `{`, `}`. Move between blank-line-delimited paragraphs. Kata: delete through the current paragraph with `d}`. [Source](https://vimhelp.org/motion.txt.html)
33. **Section starts** - Keys: `[[`, `]]`. Move to a prior or next section or first-column `{`. Kata: traverse top-level C-style blocks. [Source](https://vimhelp.org/motion.txt.html)
34. **Section ends** - Keys: `[]`, `][`. Move to a prior or next section end or first-column `}`. Kata: jump among top-level block endings. [Source](https://vimhelp.org/motion.txt.html)
35. **Unmatched braces** - Keys: `[{`, `]}`. Find the previous unmatched `{` or next unmatched `}`. Kata: navigate from nested code to enclosing block edges. [Source](https://vimhelp.org/motion.txt.html)
36. **Unmatched parentheses** - Keys: `[(`, `])`. Find the previous unmatched `(` or next unmatched `)`. Kata: locate a multiline call's bounds. [Source](https://vimhelp.org/motion.txt.html)
37. **Method boundaries** - Keys: `[m`, `]m`, `[M`, `]M`. Navigate previous or next method starts and ends. Kata: traverse methods in a class fixture. [Source](https://vimhelp.org/motion.txt.html)
38. **Matching pair** - Key: `%`. Jump between matching brackets or preprocessor branches. Kata: alternate between both ends of nested delimiters. [Source](https://vimhelp.org/motion.txt.html)
39. **Reverse match cycle** - Key: `g%`. Move through a matching-item cycle in reverse. Kata: navigate backward through `#if/#else/#endif`. [Source](https://vimhelp.org/motion.txt.html)
40. **Pattern search** - Commands: `/{pattern}`, `?{pattern}`. Search forward or backward. Kata: find exact TODO records in both directions. [Source](https://vimhelp.org/pattern.txt.html)
41. **Repeat search directionally** - Keys: `n`, `N`. Repeat the last search in the same or opposite direction. Kata: visit matches forward, then retrace them. [Source](https://vimhelp.org/pattern.txt.html)
42. **Whole-word cursor search** - Keys: `*`, `#`. Search forward or backward for the word under the cursor. Kata: visit all exact uses of an identifier. [Source](https://vimhelp.org/pattern.txt.html)
43. **Substring cursor search** - Keys: `g*`, `g#`. Search for the cursor word without word boundaries. Kata: compare `log` matches in `logger`, `catalog`, and `log`. [Source](https://vimhelp.org/pattern.txt.html)
44. **Select next or previous match** - Keys: `gn`, `gN`. Visually select the next or previous search match. Kata: change successive matches with `cgn` and `.`. [Source](https://vimhelp.org/motion.txt.html)
45. **Search offsets** - Commands: `/pat/e`, `/pat/+2`, `?pat?b`. Land at a match end or line offset. Kata: search a heading and land two lines below it. [Source](https://vimhelp.org/pattern.txt.html)
46. **Search command-line history** - Keys: `q/`, `q?`. Open editable forward or backward search history. Kata: modify and rerun an earlier complex pattern. [Source](https://vimhelp.org/pattern.txt.html)
47. **Jumplist traversal** - Keys: `<C-o>`, `<C-i>`. Move backward or forward through older cursor jumps. Kata: make several long jumps and retrace them exactly. [Source](https://vimhelp.org/motion.txt.html)
48. **Changelist traversal** - Keys: `g;`, `g,`. Visit older or newer change positions in the current buffer. Kata: edit five distant lines, then revisit each site. [Source](https://vimhelp.org/motion.txt.html)
49. **Exact local mark** - Keys: `m{a-z}`, `` `{a-z}` ``. Set and return to an exact buffer position. Kata: bookmark three edit sites and rotate among them. [Source](https://vimhelp.org/motion.txt.html)
50. **Cross-file mark** - Keys: `m{A-Z}`, `` `{A-Z}` ``. Set a global mark and jump back across buffers. Kata: connect related locations in two files. [Source](https://vimhelp.org/motion.txt.html)
51. **Linewise mark jump** - Key: `'{mark}`. Jump to a mark's line and first non-blank. Kata: compare exact backtick and linewise apostrophe jumps. [Source](https://vimhelp.org/motion.txt.html)
52. **Previous jump origin** - Keys: double-backtick, `''`. Toggle to the position or line before the latest jump. Kata: bounce between a search result and its origin. [Source](https://vimhelp.org/motion.txt.html)
53. **Last change mark** - Keys: `` `. ``, `'.`. Jump to the exact or line position of the latest change. Kata: make a remote edit and return to it. [Source](https://vimhelp.org/motion.txt.html)

58. **Local or global declaration jump** - Keys: `gd`, `gD`. Find a local or global declaration of the cursor word. Kata: navigate from a variable use to its declaration. [Source](https://vimhelp.org/motion.txt.html)

75. **Recover a small deletion** - Key: `"-p`. Restore the latest deletion shorter than a line. Kata: delete a word and line, then restore only the word. [Source](https://vimhelp.org/change.txt.html)

89. **Reuse the last search in substitution** - Commands: `/foo`, `:%s//bar/g`. Reuse the prior search pattern. Kata: inspect matches before replacing. [Source](https://vimhelp.org/pattern.txt.html)
