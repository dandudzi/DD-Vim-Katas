# Kata: Built-In Keyword Completion

## Task

Practice completing a word from the current buffer with built-in keyword completion.

## Start

Open a scratch buffer and insert:

```text
customerEmail
cus
```

Start in Insert mode after `cus` on line 2.

## End

The buffer should become:

```text
customerEmail
customerEmail
```

## Commands

Run these command steps:

```text
1. <C-n>
2. <C-y>
3. <Esc>
```
