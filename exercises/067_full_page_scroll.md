# Kata: Full Page Scroll

## Task

Practice moving forward and backward through a buffer by full pages.

## Start

Open a scratch buffer containing `Line 01` through `Line 30`, one item per line.

Start in Normal mode on line 16, column 1.

## End

The text should remain unchanged, and the cursor should return to line 16.

## Commands

Run these command steps:

```text
1. <C-f>
2. <C-b>
```
