# Kata: Format a Block with the Equal Operator

## Task

Practice deterministic built-in formatting with the `=` operator.

## Start

Open a scratch buffer with Vim filetype indentation active and insert:

```text
if ready
echo "ship"
endif
```

Start in Normal mode on the `i` in line 1.

## End

The buffer should become:

```text
if ready
  echo "ship"
endif
```

## Commands

Run these command steps:

```text
1. =G
```
