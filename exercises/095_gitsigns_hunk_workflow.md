# Kata: Gitsigns Preview Hunk

## Task

Practice previewing a prepared changed Git hunk with Gitsigns.

## Start

Open an existing tracked practice file with one changed line in its Git worktree.

Start in Normal mode inside the changed hunk.

## End

The file should remain unchanged, and the Gitsigns hunk preview should be visible.

## Commands

Run these command steps:

```text
1. <leader>ghp
```
