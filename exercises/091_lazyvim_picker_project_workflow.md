# Kata: LazyVim Find Files Picker

## Task

Practice opening an existing project file with LazyVim's root file picker.

## Start

Open a scratch buffer and insert:

```text
practice_js_ts_filetypes.ts
```

Start in Normal mode on the `p` in line 1.

## End

The current buffer should be `practice_js_ts_filetypes.ts`.

## Commands

Run these command steps:

```text
1. <leader>ff
2. practice_js_ts_filetypes
3. <CR>
```
