# Kata: Generate an Incrementing Sequence

## Task

Practice turning repeated numbers into an incrementing sequence with Visual block mode and `g<C-a>`.

## Start

Open a scratch buffer and insert:

```text
item_01
item_01
item_01
item_01
```

Start in Normal mode on the `1` in the first line's number.

## End

The buffer should become:

```text
item_02
item_03
item_04
item_05
```

## Commands

Run these command steps:

```text
1. <C-v>jjj
2. g<C-a>
```
