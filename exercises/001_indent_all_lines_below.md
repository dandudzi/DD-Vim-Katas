### Indent all lines below

Use `>G` to indent all lines till the end of file.

Change this:

```javascript
var foo = 1;
var bar = 'a';
var foobar = foo + bar;
var foobar = foo + bar;
var foobar = foo + bar;
var foobar = foo + bar;
var foobar = foo + bar;
```

to this:

```javascript
var foo = 1
var bar = 'a'
var foobar = foo + bar
  var foobar = foo + bar
  var foobar = foo + bar
    var foobar = foo + bar
    var foobar = foo + bar
```
