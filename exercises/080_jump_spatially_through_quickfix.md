# Kata: Navigate a Prepared Quickfix List

## Task

Practice moving through a prepared quickfix list with LazyVim's quickfix mappings.

## Start

Open `practice_152_spatial_quickfix.txt` with a prepared quickfix list containing its three `TODO` lines.

Start in Normal mode on the `T` in the first `TODO` at line 2, column 1.

## End

The quickfix list should be visible, and the cursor should return to the first `TODO` at line 2.

## Commands

Run these command steps:

```text
1. <leader>xq
2. ]q
3. ]q
4. [q
5. [q
```
