# Kata: Noice Message History

## Task

Practice opening and closing Noice message history with LazyVim mappings.

## Start

In LazyVim with Noice enabled, wait until Neovim has displayed at least one
message. Start in Normal mode; the cursor position does not matter.

## End

Noice message history should be closed after you inspect it.

## Commands

Run these command steps:

```text
1. <leader>snh
2. q
```
