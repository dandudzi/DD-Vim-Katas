### Insert at the end of line

Use `>G` to indent all lines till the end of file.

to this:

````javascript
var foo = 1
var bar = 'a'
var foobar = foo + bar
  var foobar = foo + bar
  var foobar = foo + bar
    var foobar = foo + bar
    var foobar = foo + bar
    ```


```javascript
var foo = 1;
var bar = 'a';
var foobar = foo + bar;
var foobar = foo + bar;
var foobar = foo + bar;
var foobar = foo + bar;
var foobar = foo + bar;
````
