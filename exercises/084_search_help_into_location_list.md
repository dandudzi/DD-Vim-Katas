# Kata: Search Vim Help with the LazyVim Picker

## Task

Practice finding a Vim help topic through LazyVim's help picker.

## Start

Open a scratch buffer and insert:

```text
quickfix
```

Start in Normal mode on the `q` in line 1.

## End

The current window should show a help buffer for a quickfix-related topic.

## Commands

Run these command steps:

```text
1. <leader>sh
2. quickfix
3. <CR>
```
