# Kata: Build and Navigate a Split Layout

## Task

Practice creating, navigating, resizing, and closing split windows with LazyVim and native window keys.

## Start

Start in Normal mode in any buffer with one window open. The cursor position
does not matter.

## End

Only one window should remain.

## Commands

Run these command steps:

```text
1. <leader>|
2. <leader>-
3. <C-w>h
4. <C-w>l
5. <C-w>j
6. <C-w>=
7. 8<C-w>_
8. <C-w>o
```
