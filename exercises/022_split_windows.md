# Kata: Build and Navigate a Split Layout

## Task

Practice creating, navigating, resizing, and closing split windows with LazyVim and native window keys.

## Start

Open a scratch buffer and insert:

```text
split practice
```

Start in Normal mode on the `s` in line 1 with one window open.

## End

Only one window should remain, and the buffer should still contain:

```text
split practice
```

## Commands

Run these command steps:

```text
1. <leader>|
2. <leader>-
3. <C-w>h
4. <C-w>l
5. <C-w>j
6. <C-w>=
7. 8<C-w>_
8. <C-w>o
```
