# Kata: Half Page Scroll

## Task

Practice moving the cursor and viewport with half-page scrolls.

## Start

Open a scratch buffer containing `Line 01` through `Line 20`, one item per line.

Start in Normal mode on line 9, column 1.

## End

The text should remain unchanged, and the cursor should return to line 9.

## Commands

Run these command steps:

```text
1. <C-d>
2. <C-u>
```
