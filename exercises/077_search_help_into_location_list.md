# Kata: Send Vim Help Search Results to Quickfix

## Task

Practice searching Vim help and sending the matching topics to the quickfix list.

## Start

Start in Normal mode in LazyVim. The cursor position does not matter.

## End

The quickfix list should be visible and contain the help matches filtered by
the `quickfix` query. The number of entries may vary between Vim and Neovim
versions.

## Commands

Run these command steps:

```text
1. <leader>sh
2. quickfix
3. <C-q>
```
