# Kata: Filter an Existing Quickfix List

## Task

Practice opening a prepared quickfix list in LazyVim's picker and filtering its entries interactively.

## Start

Open `practice_154_quickfix_filter.txt` with a prepared quickfix list containing all five `TODO` lines.

Start in Normal mode on line 1, column 1.

## End

The quickfix picker should show only entries whose text contains `core`.

## Commands

Run these command steps:

```text
1. <leader>sq
2. core
```
