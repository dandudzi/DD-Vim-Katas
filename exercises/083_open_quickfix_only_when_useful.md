# Kata: Toggle a Prepared Quickfix List

## Task

Practice opening and closing a non-empty prepared quickfix list with LazyVim.

## Start

Open `practice_156_app.txt` with a prepared quickfix list containing the two entries from `practice_156_fail.txt`.

Start in Normal mode on line 1, column 1.

## End

The quickfix window should be closed, and the current buffer should be `practice_156_app.txt` on line 1.

## Commands

Run these command steps:

```text
1. <leader>xq
2. <leader>xq
```
