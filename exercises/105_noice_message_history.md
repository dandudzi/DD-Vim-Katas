# Kata: Noice Message History

## Task

Practice opening and closing Noice message history with LazyVim mappings.

## Start

Open a scratch buffer in LazyVim with Noice and insert:

```text
noice buffer stays unchanged
```

Start in Normal mode on the `n` in line 1 after Neovim has displayed at least one message.

## End

The buffer should remain unchanged after inspecting and closing Noice history.

## Commands

Run these command steps:

```text
1. <leader>snh
2. q
```
