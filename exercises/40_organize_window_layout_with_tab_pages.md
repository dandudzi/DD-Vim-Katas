### Organize window layout with tab pages

All about tab pages

# tabs

`:tabc` - close the current tab
`:tabo` - keep the active page, close all tabs
`:tabe 01..` - Open the 01 markdown file in a new tab
`gt` - move to next tab
`gT` - move to previous tab
`:e!` - relode file from disk
`:wa` - write all fiels
`:qa!` - quit all files
`<s-h>` - next tab
`<s-l>` - prev tab

# windows

`:sp 02` - open the 02 markdown file in a horizontal split window
`:vsp 02` - open the 02 markdown file in a horizontaly split window
`<C-w>T` - move the current window into its own tab
`<C-w>s` - split horizontaly
`<C-w>h/j/k/l` - move between windows
`<C-w>c` - close window
`<C-w>o` - close all windows and keep only acitve
`<C-w>=` - eqalise size of all windows
`<C-w>_` - maximise heigh of the active window
`<C-w>|` - masimise width of the window
`[N]<C-w>_` - set height to N rows of the active window
`[N]<C-w>|` - set widith to N columsn  width of the window
