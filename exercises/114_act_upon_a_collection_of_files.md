# Kata: Navigate a Collection of Buffers

## Task

Practice moving through a prepared buffer collection with LazyVim buffer mappings.

## Start

Prepare three listed buffers named `alpha.txt`, `beta.txt`, and `gamma.txt`.

Start in Normal mode in `alpha.txt` on line 1, column 1.

## End

The current buffer should be `beta.txt`.

## Commands

Run these command steps:

```text
1. ]b
2. ]b
3. [b
```
